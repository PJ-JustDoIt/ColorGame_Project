ColorGame features :
---------------------
The Game displays RGB(#,#,#) on top, which shows the amount of Red,Green,Blue paints mixed together to create a color. 
User guesses which among the 6 tiles (HARD option ) or 3 tiles (EASY option ) is closest to this created color .
User taps his/her guess tile with the mouse.
If User wants to "Pass" the round in the game, he/she can click on "NEW COLORS"


App uses : 
----------
DOM manipulation / events , javascript, html, css
